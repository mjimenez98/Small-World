# Small-World
Winter 2018, COMP 345 Project.
# How clone repo ?

For clone the repo, you have to run : 

```https://github.com/mjimenez98/Small-World.git```

Install cmake : 

Ubuntu : `apt install cmake`
Arch Linux : `yay cmake` or `pacman -S cmake`

Run cmake : 

``` cd Small-world
cmake . ```
